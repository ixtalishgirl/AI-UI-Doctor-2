All Tools


    

      Element Fix
      
Pick an element with "Select Element" above, then describe the change in plain English. The AI writes minimal CSS or JS, you preview live, and revert anytime.

      
      
    

    

      Smart Page Scan
      
Scans the whole page for broken images, overlapping text, low color contrast, and console errors. The AI suggests one CSS fix for everything found.

      
    

    

      Chrome Debugger
      
Attaches Chrome DevTools to the page. Captures hidden runtime exceptions, memory leaks, DOM/listener counts, and failing API calls (with request and response bodies).

      

        
        
      

      

      

      

      
Tip: attach first, then reload the page to capture startup errors.

      

    

    

      Hidden State Reader
      
Shows everything hidden on the page: cookies (including HttpOnly), localStorage, sessionStorage, hidden form inputs, JSON data blocks, internal IDs, and page globals like Next.js or Redux state.

      
    

    

      JWT & Auth Token Finder
      
Scans cookies, storage, and hidden inputs for JWT-format tokens. Decodes the header and payload so you can check for sensitive data, weak algorithms (alg: none), or missing expiry.

      
    

    

      SQL Injection Tester
      
Sends 45+ SQL injection payloads to your API endpoint and looks for 17 database error signatures (MySQL, Postgres, Oracle, SQLite, MSSQL).

      
      
      
      
      
      
      
      
      
    

    

      XSS Form Tester
      
Lists every form on the page. Pick one and the tool injects 12+ unique-marker XSS payloads to check if the server escapes them or lets them through.

      
      
      
    

    

      API Endpoint Discovery (Hidden Routes)
      
Probes 400+ common hidden paths on your site — admin panels, debug endpoints, exposed config files like .env and .git, Spring Boot actuators, Swagger docs, backups, log files.

      
      
      
      
      

    

    

      Automated Payload Fuzzer (10 Categories)
      
Sends a library of attack payloads (SQL, XSS, command injection, path traversal, SSRF, NoSQL, template injection, LDAP, SSTI, JSON proto-pollution, CRLF, Host header) to one parameter on your endpoint.

      
      
      
      
      
      
      
      
      
      
      
    

    

      Session & Token Replay Audit
      
Lists every cookie set on your site and checks the security flags: HttpOnly, Secure, SameSite.

      
    

    

      Logic Flaw Tester (Price / Balance / Quantity Bypass)
      
Sends multiple price or balance variants (negative numbers, zero, decimals, scientific notation, integer overflow, type confusion) to your purchase or balance endpoint.

      
      
      
      
      
      
      
    

    

      CORS Misconfiguration Tester
      
Sends a request with a fake malicious Origin header. If your server reflects it back, any other website can read your API responses on behalf of logged-in users.

      
      
      
    

    

      Rate Limit Tester (Brute-Force Protection)
      
Sends N rapid requests to one endpoint and reports how many succeeded before the server started blocking or returning 429. Supports up to 1000 concurrent requests.

      
      
      
      
      
      
      
      
      
    

    

      Open Redirect Tester
      
Tries 60+ common redirect parameter names (?next=, ?url=, ?redirect=, ?return=, and more) with a malicious target.

      
      
      
    

    

      Header Manipulation
      
Add, remove, or spoof any HTTP request or response header. Rules are session-only and disappear when Chrome closes.

      
      
      
      
      
      
      
      
      

        
        
      

      

    

    

      Disable CSP (one-click)
      
Strips Content-Security-Policy and X-Frame-Options headers from all responses. Useful to test whether your CSP is actually blocking attacks. Reload the page after enabling.

      
    

    

      Global CSS Refactor (AI suggestion)
      
The AI looks at your full page structure and suggests site-wide CSS variables and improvements. Apply Live to preview, Revert to undo.

      
    

    

      Inject Global CSS
      
Paste raw CSS here and inject it into the current page immediately. Useful for quick visual overrides.

      
      
    

    

      Script Manager
      
List all scripts on the page. Select one to replace its content — useful to stub analytics or test behavior without the original script running.

      
      

      
      
      
      
      
    

    

      🚨 Balance Attack (GOD TIER — 35+ amounts)
      
Sends 35+ amount variants (negative, zero, scientific notation, type confusion, NoSQL operators, integer overflow) as real authenticated requests to your balance/purchase endpoint. Also fires 20 parallel requests to test race conditions.

      
      
      
      
      
      
      
      
      
      
      
      
      
    

    

      🔓 JWT Auth Bypass (32 variants)
      
Finds JWT tokens in your storage, then forges 32 attack variants (alg=none, role=admin, kid=SQLi/SSRF/path-traversal, RS256→HS256 confusion, god-claims, PASETO confusion) and sends each as a Bearer token.

      
      
      
    

    

      JWT Cracker (200+ wordlist)
      
Brute-forces the HMAC-SHA256 secret key of any JWT you paste below using a 200+ entry wordlist of common secrets.

      
    

    

      Request Replay (Spoof IP & Headers)
      
Replay any request with custom headers and optionally a spoofed X-Forwarded-For IP to bypass IP allowlists and test admin-area access control.

      
      
      
      
      
      
      
      
      
      
      
    

    

      Wayback & URLscan Miner
      
Queries web.archive.org and urlscan.io for every URL ever indexed for your domain. Up to 300 Wayback URLs + 200 URLscan results.

      
      
      
    

    

      Dependency CVE Check
      
Detects JS library versions loaded by the page and queries OSV.dev for known CVEs. Up to 12 libraries checked.

      
    

    

      Prototype Pollution Test
      
Injects __proto__[polluted]=YES and constructor[prototype][polluted2]=YES via URL params and checks if Object.prototype gets polluted in the page.

      
      
      
    

    

      HTTP Request Smuggling
      
Sends 4+ HTTP smuggling probes (TE:chunked + CL conflict, duplicate TE, CL:0 with second request body, duplicate Host) to detect desync vulnerabilities between your reverse proxy and backend.

      
      
      
    

    

      WebSocket Fuzzer
      
Installs a monitor on the page that logs every WebSocket frame. Then send any payload to all live connections.

      
      
      
      
    

    

      Source Map Extractor
      
Finds sourceMappingURL comments in your JS files and tries to download the .map files. If exposed, attacker can recover your original source code.

      
    

    

      Subdomain Finder (200+ prefixes)
      
Mines Certificate Transparency (crt.sh) for all subdomains, then brute-forces 200+ common prefixes via DNS-over-HTTPS, and probes each live subdomain for its HTTP status and server.

      
      
      
    

    

      Subdomain Takeover Check
      
Checks each subdomain found by the Subdomain Finder against 20+ dangling-service fingerprints (GitHub Pages, S3, Heroku, Vercel, Netlify, Azure, and more). Auto-uses last subdomain_finder run.

      
    

    

      WAF / CDN Detector (30+ signatures)
      
Sends a deliberately suspicious payload and detects 30+ WAFs/CDNs including Cloudflare, AWS WAF, Akamai, Sucuri, Imperva, F5, ModSec, Wordfence, Barracuda, Fastly, Vercel, Azure Front Door, and more.

      
    

    

      Blind SQLi (Time-Based)
      
Sends 24+ time-based payloads (MySQL SLEEP, Postgres pg_sleep, MSSQL WAITFOR, Oracle DBMS_PIPE, SQLite randomblob) and measures response times to detect blind SQL injection without visible errors.

      
      
      
      
      
      
      
      
      
    

    

      OAuth Bypass Tester
      
Probes OAuth redirect_uri open redirect, state CSRF (sends identical state twice), and token leakage in URL fragment or query string.

      
      
      
    

    

      XXE Injection (14 vectors)
      
Sends 14 XML External Entity payloads to an XML-accepting endpoint: /etc/passwd read, win.ini, AWS metadata SSRF, GCP metadata, localhost SSRF, XInclude, PHP filter, OOB, Billion Laughs DoS, and more.

      
      
      
      
      
    

    

      SSRF Advanced (35+ cloud metadata targets)
      
Probes 35+ SSRF targets via your endpoint: AWS/GCP/Azure instance metadata, Docker socket, Kubernetes API, Redis, Elasticsearch, and internal service discovery.

      
      
      
      
      
    

    

      2FA / MFA Bypass (23 variants)
      
Tests 23 known 2FA/MFA bypass techniques: null/empty/object/array/boolean code value, step injection, backup-code brute-force, trusted_device flag, NoSQL operator injection, and more.

      
      
      
      
      
    

    

      DOM Monitor (Live Element Watcher)
      
Installs a MutationObserver on the page that watches for DOM changes to balance/wallet elements in real time. Reports any changes to the Live Terminal.

      
      
      
    

    

      Tech Fingerprint
      
Detects server technology, framework, hosting provider, CDN, and security products from response headers and page body content.

      
      
      
    

    

      JS Secret Scanner (80+ patterns)
      
Downloads all JS files used by the page and regex-matches against 80+ secret patterns: AWS/Google/Stripe/GitHub/Slack/OpenAI/Anthropic keys, JWTs, private keys, database URIs, and more.

      
    

    

      CSRF Token Check
      
Lists all forms on the page and reports which POST/PUT/PATCH/DELETE forms are missing a CSRF token field.

      
    

    

      TLS / HSTS Audit
      
Checks for HTTPS enforcement (HTTP→HTTPS redirect), HSTS header, and preload eligibility. Also checks if cookies are set with the Secure flag.

      
      
      
    

    

      Clickjacking Defense Test
      
Checks X-Frame-Options and Content-Security-Policy frame-ancestors directive to verify the page cannot be embedded in an iframe by a malicious site.

      
      
      
    

    

      Security Headers Checker
      
Audits 9 critical security headers (CSP, HSTS, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy, COOP, COEP, CORP) and flags 6 server-leak headers.

      
      
      
    

    

      IDOR Tester (Object Access)
      
Takes a URL with a numeric ID in the path or query string, increments/decrements it by ±5 and ±10, and checks if the server returns another user's data.

      
      
      
    

    

      Mixed Content / SRI Audit
      
Finds http:// resources loaded on an https:// page, and third-party scripts missing Subresource Integrity (SRI) integrity= hashes.

      
    

    

      GraphQL Introspection Probe
      
Probes 25+ common GraphQL paths and checks if introspection is enabled (a major schema leak). Also tries common mutation operations.

      
      
      
    

    

      Race Condition Tester
      
Fires N parallel requests at one endpoint simultaneously to detect race conditions in claim/redeem/withdraw/transfer-style actions. Up to 1000 concurrent requests.
